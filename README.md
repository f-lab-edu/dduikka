# 뛰까

## 프로젝트 소개

위치 기반 날씨를 제공하는 러너들을 위한 커뮤니티, 뛰까 입니다.

사용자는 자신이 저장한 위치를 기반으로 날씨와 대기정보를 조회할 수 있습니다.

오늘 자신이 뛸지, 말지 투표할 수 있으며 투표한 내역은 익명 채팅에 표현됩니다.

익명 채팅을 통해 같이 달리는 친구들과 동질감을 느껴보세요!
